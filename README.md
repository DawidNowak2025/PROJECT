GROW - Garden Resource Organisation and Watering
This is the multi-view version of the Smart Garden Tracker project.

The application is split into clear sections:

Dashboard
Plants
Garden Map
Watering
Activity Log
This makes the project look and work more like a real garden tracking system, not only a simple notepad.

Main Features:

Multi-view single page application layout
Dashboard with clickable summary cards
Add, edit, delete and water plants
Plant photo and separate location photo
Full-screen image preview after clicking a photo
Garden zones and clickable garden map
Add custom garden zones
Watering tracker with overdue and upcoming plants
Plant health status tracking
Plant care recommendations
Plant history notes
Automatic activity log
Local JSON file storage

Technology Stack:

Frontend: React.js with simple CSS
Backend: Node.js and Express.js
Storage: Local JSON files
Libraries: Axios and Multer

How to Run:
Backend
cd backend
npm install
npm start
Backend runs on:

http://localhost:5000
Frontend
Open a second terminal:

cd frontend
npm install
npm start
Frontend runs on:

http://localhost:3000
